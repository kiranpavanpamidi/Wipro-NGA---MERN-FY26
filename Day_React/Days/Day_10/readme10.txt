Day 1: React Basics & Component Fundamentals

Title: “BookVerse: Welcome Page and Featured Books Section”

### Details
A simple React project demonstrating the fundamentals of React, including components, props, state, JSX, event handling, controlled components, and dynamic UI rendering.

### Futures
View popular books
Switch between Grid and List view
Search books using a controlled input box
Understand the fundamentals of React components & props
