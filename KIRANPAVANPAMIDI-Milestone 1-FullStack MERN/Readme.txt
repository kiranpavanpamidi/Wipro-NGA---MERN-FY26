#Overview
This project implements a functional, responsive, and data-driven E-Commerce Product Management Interface using 'HTML5, CSS3, Bootstrap, JavaScript (ES6), and TypeScript'.  
It is divided into three main User Stories, each focusing on a different area of front-end development.

@ How to Run the Project

#Requirements
- Node.js installed  
- TypeScript compiler (`npm install -g typescript`)  
- Browser (Chrome, Edge, Firefox)  
- Code editor (VS Code preferred)

#Steps to Execute

> For User Story 1
1. Open `userstory1/index.html` in a browser.  
2. Test the responsive layout and newsletter form validation.

> For User Story 2
1. Open `userstory2/products.html` in a browser.  
2. Wait for products to load from `products.json`.  
3. Use the dropdown menu to filter by category.

> For User Story 3
1. Navigate to `userstory3/` in your terminal.  
2. Compile TypeScript to JavaScript:  
   bash
   tsc productManager.ts
   node .\productManager.ts
