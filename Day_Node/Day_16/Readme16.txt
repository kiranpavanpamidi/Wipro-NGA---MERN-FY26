Day 16 — Node.js & Basics

### Details
This module introduces the fundamentals of Node.js, npm, and working with backend JavaScript.

### Features
Challenge 1 — Node Fundamentals
Logs Node version, file name, and directory
Prints a message every 3 seconds
Stops after 10 seconds (bonus)

Challenge 2 — npm Usage
Initializes a Node project
Installs external npm packages
Displays a colorful ASCII banner using figlet + chalk
Includes npm start script

Challenge 3 — CLI App
Accepts a username from command line
Prints greeting with current date/time
Uses moment for formatted date (bonus)
